Konum izni nasıl açılır?

Tarayıcınızda site için konum izni kapalıysa uygulama otomatik konum alamaz. Aşağıdaki kısa adımları izleyin:

  • Chrome / Edge: Adres çubuğundaki kilit simgesine tıklayın → Site ayarları → Konum → Allow.
  • Firefox: Adres çubuğundaki kilit → Permissions → Location → Allow.
  • Safari (macOS): Safari → Settings for This Website → Location → Allow.

Değişiklikten sonra sayfayı yenileyin ve tekrar Konumumu Kullan butonuna basın.
